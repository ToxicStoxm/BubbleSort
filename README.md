# BubbleSort

Simple CLI bubble sort implementation

## Using the software
### Requirements:
  - meson
  - opencv4
### Building:
  #### 1. Clone this repositry
      git clone https://github.com/ToxicStoxm/BubbleSort/
  #### 2. Cd into the project root
      cd BubbleSort
  #### 3. Run meson
      meson setup ./build
  #### 4. Run ninja
      ninja -C ./buiild
  #### 5. Run the program
      ./build/src/bubblesort
